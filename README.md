## Description

Cart API test service for FlipFIT.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# test coverage
$ npm run test:cov
```

## Documentation of endpoints

- navigate to https://127.0.0.1:3000/docs
- use 'Try it out' to test requests
